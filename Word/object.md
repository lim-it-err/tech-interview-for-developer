## 인터넷 정의
데이터와 행위의 집합체로서, 데이터와 이의 행위(함수)를 하나로 묶음으로서 상호 대상의 단위로 구성.

## 재구성
- 다른 객체 A, B에 대해
- A, B는 식별 및 구분이 가능하여
  - 독립적인 하나의 단위로 인식 가능하고
  - 개수를 셀 수 있고
  - 생성 시점을 알 수 있고,
- 상태와 행동이 묶인 실체이다. 

## 성질
### 상태 + 행위 관련
- 캡슐화 : 묶인 상태와 행동을 외부 인터페이스만을 통해 통신한다.
- 상속성 : 기존 객체를 재사용하여, 구체적인 객체를 구성함.
### 행위 관련
- 다형성 : 같은 메서드에 대해, 다른 행위를 할 수 있도록 지원
